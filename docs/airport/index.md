# 机场

机场是指提供节点的商家。它们以订阅链接的形式向你提供若干数量的节点，你把订阅链接导入到代理客户端，代理客户端连接这些节点便可翻墙。

### 如何选择

机场的数量要远远大于VPN。他们的质量层次不齐，好坏主要取决于节点速度是否快，以及是否可以解锁一些特定的网站。机场大多提供免费试用，好坏只用试用了才知道。

机场有跑路的风险，并且他们的质量可能时好时坏，所以谨慎购买年付套餐。

### 隐私与安全

##### 机场能知道我访问了哪些网站吗
取决于你怎么用机场。如果你只用一家机场，那么该机场能够知道你访问的网站。如果你使用机场来做链式代理，那么只有落地节点可以知道你访问了哪些网站，前置的机场无法知道你访问了什么网站。你可以采用“前置机场+自建落地节点”的方式来保护隐私。

##### 机场能知道我的账号密码或者我在网站干了什么吗
对于绝大多数网站，机场无法知道，因为它们都使用了加密（HTTPS）。机场看到的是加密的数据，它无法破解。对于极少数使用HTTP的网站，机场能知道。访问这些HTTP网站时，浏览器会在地址栏明确告知你不安全。
