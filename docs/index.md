# 简介

小白翻墙指南 - 科学上网教程

### 什么是翻墙

一些国外网站无法在大陆访问，是因为“防火墙”在阻止你访问它们。突破防护墙的封锁来访问那些网站，就是翻墙，又称科学上网。

### 主流翻墙方式

主流翻墙方式有以下三种：

- VPN：使用VPN公司提供的软件。
- 机场：导入机场提供的订阅链接到代理客户端来翻墙。
- 自建：自己购买VPS（服务器），在VPS上部署代理软件如V2ray，通过代理客户端连接VPS来翻墙。

这三种方式背后的原理是一样的：VPN软件或者代理客户端加密你的网络数据，将加密后的数据发送到一个没有被防火墙封禁的服务器，该服务器会将数据发送给你想要访问的网站，并且把网站返回的数据回传给你。

还有一种小众的翻墙方式：使用海外手机卡。这种方式价格较贵，但是它能够做到让监管者无法知道你在翻墙。

### 为什么要科学上网

除了可以访问国内被封禁的网站外，还有一个用途：加速你的网络，很多网站即便没有被封禁，直连访问起来速度会很慢。

### 翻墙会被知道吗
如果你使用主流的翻墙方式，有可能会被有关部门知道。使用一些具备伪装效果的代理协议可以降低概率。对于绝大多数人来说，无需担心这个问题。

### 翻墙违法吗
有说违法的，也有说不违法的。重要的是，因为单纯翻墙而被警察找的概率几乎为零。
