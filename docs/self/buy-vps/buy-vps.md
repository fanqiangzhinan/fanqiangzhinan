# 选购建议

VPS商家有很多，知名大厂如谷歌云，AWS（亚马逊），Oracle
Cloud（甲骨文）。知名小厂如搬瓦工。

大厂VPS价格较高，性能也好，支持免费试用。管理起来很方便，支持随时更换IP，更换地点。它们往往采取按量计费，用多少付多少。

小厂VPS价格较低，但是它们一般不支持试用，很多也不支持更换IP，更换地点。采取每月/每年固定费用。
